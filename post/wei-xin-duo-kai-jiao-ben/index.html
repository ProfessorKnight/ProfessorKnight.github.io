# 微信多开脚本

2020-03-26 2 min read 
  推荐语：

  电脑上同时登陆多个微信最简单的方法。

适应操作系统：windows

脚本位置：会员群 -->2. 成人-->效率提升

下面的批处理文件可以实现双开微信，如果想开三个，中间就再加一句命令。如果微信程序不是默认安装在C盘的，就将文件中的路径修改为正确的安装路径。或者卸载微信重新默认安装。

将下面的文本保存再记事本中，保存成.bat的格式，运行即可实现多开（运行时请关闭开着的微信程序）。

可以实现双开的bat文件已经放入网盘群中，会员自取。

@echo off

start /d "C:\Program Files (x86)\Tencent\WeChat\" WeChat.exe

start /d "C:\Program Files (x86)\Tencent\WeChat\" WeChat.exe

exit
