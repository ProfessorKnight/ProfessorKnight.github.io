# 时间管理-Randy Pausch

2020-03-26 1 min read 
  时间管理可以让你安排出更多的时间，陪伴你爱的人。

  有一天你可能会发觉，你拥有的比你想象的少。

相关文件路径：会员群-->3.成人-->我的推荐-视频类

第一次听Randy Pausch教授的演讲已经是十几年前了，那时的他，终究还是没能战胜病魔。

听了他的演讲之后，我开始着手认真对待自己的时间，虽然这些年自己并没有做到自己满意的效果，但是仍旧收益良多。

核心内容如下（这里我就不展开了，感兴趣的可以自己搜索，日后的分享中还会设计这些内容）：

  1. TODO list
  2. 2-8法则
  3. Cover's four-quadrant TODO

里面一些提示也很重要：电脑显示器两个以上（谁用谁知道）

和其它所有事情一样，时间管理中最重要的事情就是执行，没有做过时间管理的人，如果想开始尝试，那就马上迈出你的第一步，打扫干净房间。
